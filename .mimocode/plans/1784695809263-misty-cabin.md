# Email Verification Implementation Plan

## Goal
Add email verification after sign-up. User must verify email before accessing the app.

## Architecture

### New States & Events

**auth_state.dart** — add:
- `EmailNotVerified(User user)` — user logged in but email not verified

**auth_event.dart** — add:
- `SendVerificationEmail()` — resend verification email
- `CheckEmailVerified()` — check if user clicked the link

### Changes by File

#### 1. `lib/services/auth_service.dart`
Add two methods:
```dart
Future<void> sendEmailVerification() async {
  await _firebaseAuth.currentUser?.sendEmailVerification();
}

Future<void> reloadUser() async {
  await _firebaseAuth.currentUser?.reload();
}
```

#### 2. `lib/bloc/auth/auth_bloc.dart`
- Register new event handlers: `on<SendVerificationEmail>`, `on<CheckEmailVerified>`
- **`_onSignUp`**: after successful sign-up → call `authService.sendEmailVerification()` → emit `EmailNotVerified(user)`
- **`_onUserChanged`**: check `user.emailVerified` — if false → emit `EmailNotVerified`, if true → emit `Authenticated`
- **`_onSignIn`**: check `credential.user.emailVerified` — if false → emit `EmailNotVerified`, if true → emit `Authenticated`
- **`SendVerificationEmail`**: call `authService.sendEmailVerification()`
- **`CheckEmailVerified`**: call `authService.reloadUser()`, then re-check `emailVerified`

#### 3. `lib/main.dart` — `AuthGate`
Add handling for `EmailNotVerified` state in builder:
```dart
if (state is EmailNotVerified) {
  return EmailVerificationPage(user: state.user);
}
```

#### 4. New file: `lib/pages/authentication/email_verification_page.dart`
Simple page with:
- Icon (email)
- Title: "Verify your email"
- Subtitle: "We sent a verification link to {email}"
- "Resend Email" button → dispatches `SendVerificationEmail`
- "I've Verified" button → dispatches `CheckEmailVerified`
- "Sign Out" button → dispatches `SignOutRequested`

#### 5. `lib/pages/authentication/sign_up_page.dart`
- Remove the auto `Navigator.pop` after 1 second (let AuthGate handle navigation via state)
- After `SignUpRequested` is dispatched, AuthBloc emits `EmailNotVerified` → AuthGate renders verification page automatically

## Flow

```
Sign Up → AuthBloc._onSignUp
  → Firebase creates user
  → sendEmailVerification()
  → emit(EmailNotVerified(user))
  → AuthGate builder shows EmailVerificationPage

User clicks link in email
  → User returns to app, taps "I've Verified"
  → CheckEmailVerified event
  → reloadUser() → check emailVerified
  → if true → emit(Authenticated) → AuthGate shows MyHomePage
  → if false → show error "Email still not verified"
```
